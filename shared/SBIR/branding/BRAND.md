## Typography

### Primary Font
**JetBrains Mono** — Monospace typeface for all branding

```css
font-family: 'JetBrains Mono', 'Monaco', 'Menlo', 'Courier New', monospace;
```

### Fallback Stack
`Monaco` → `Menlo` → `Courier New` → `monospace`

### Weights
- **600 (SemiBold)** — Logo, headings
- **400 (Regular)** — Body text, taglines

---

## Usage Rules

### Do
- Use on dark backgrounds (#0C0C0C or darker)
- Maintain the underscore cursor
- Keep the monospace aesthetic
- Use sufficient contrast

### Don't
- Stretch or distort the logo
- Change the green color
- Remove the underscore
- Use on busy backgrounds
- Add effects (shadows, gradients, 3D)
